# Customer App With Imported Menu

This version imports the attached menu into the customer ordering app and groups items into categories.

## Categories included
- Breakfast Burritos
- Savory Burritos
- French Specialties
- Mexican Favorites
- Quesadillas
- Sides
- House Specialties
- Desserts
- Beverages

## Required environment variable
- NEXT_PUBLIC_ADMIN_API_URL=https://your-admin-app.vercel.app
